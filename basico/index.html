Encabezados

Los encabezados (h1 a h6) se utilizan para definir títulos y subtítulos en una página. h1 es el más importante y h6 el menos importante.

Párrafos

La etiqueta p se utiliza para definir bloques de texto o párrafos. Es ideal para contenido descriptivo o explicativo.

Este es un ejemplo de un párrafo simple.

Puedes usar varios párrafos para dividir el contenido en bloques más fáciles de leer. Cada párrafo se representa con una nueva etiqueta p.
